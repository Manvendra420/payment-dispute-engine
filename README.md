# Payment Dispute Engine

## Overview
A workflow-driven system to manage card payment disputes, chargebacks,
evidence submission, arbitration, and analytics.

## Core Features
- Dispute lifecycle engine
- SLA & deadline enforcement
- Evidence management
- Rules-based decision engine
- Audit-safe event log

## Architecture
(Event-driven, long-running workflows)

## Tech Stack
- Backend: Python, FastAPI
- DB: PostgreSQL
- Queue: Azure Service Bus
- Storage: Azure Blob Storage
- Frontend: Next.js

## Getting Started
docker-compose up
